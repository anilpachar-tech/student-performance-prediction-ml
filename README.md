# 🎓 Student Performance Prediction Using Machine Learning
Machine Learning project to predict student academic performance using multiple regression models with hyperparameter tuning and data analysis.
This project predicts student academic performance based on demographic and study-related factors using machine learning algorithms.

## 🚀 Features

- Data preprocessing and visualization
- Feature engineering
- Multiple machine learning models
  - Linear Regression
  - Decision Tree
  - Random Forest
  - Support Vector Regression
- Model comparison and evaluation
- Cross-validation
- Hyperparameter tuning using GridSearchCV
- Feature importance analysis

## 📊 Dataset

Dataset contains information about students including:

- Gender
- Parental education
- Lunch type
- Test preparation course
- Math, Reading, and Writing scores

Target variable:
- Average Score

## 🧠 Machine Learning Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Hyperparameter Optimization
7. Prediction

## 🏆 Results

Random Forest achieved the best performance with high R² score after hyperparameter tuning.
